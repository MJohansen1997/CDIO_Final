Velkommen tilbage til Rødovrevej Pharmaceutics

Hvad står vi for?

Hos Rødovre Pharmaceutics går vi op i kvalitet, både hos vores eget arbejde, men også hvordan vores patienter bliver behandler. Vi striver efter bedre service og hvordan vi kan forbedre vores recepter, så de hjælper dig endnu mere i din hverdag. Herunder modtager vores råvarer hver uge, for at sørge at vores produktioner er friske og altid er i god kvalitet. Vores hold arbejder på hver dag, at udvikle nye produkter til at forbedre den medicinske verden.
